**Tenzies Game**

A simple and fun Tenzies game built with React.The goal of the game is to roll until all dice show the same number. You can click on individual dice to hold their value between rolls.

How many rolls did you take? : https://funwithtenzies.vercel.app/

**Features**
Roll 10 dice at once
Hold dice between rolls
Win detection when all dice match
Roll counter
Timer tracking
Clean and responsive UI
Confetti animation on win

**Built With**
React,
Vite,
JavaScript,
CSS,
Nano ID,
React Confetti
